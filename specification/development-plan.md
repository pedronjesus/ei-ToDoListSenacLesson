# Plano de Desenvolvimento por Fases

## Fase 0 — Configuração Inicial

* Criar o repositório do projeto
* Adicionar arquivos base (HTML, CSS e JS)
* Garantir que a aplicação abre corretamente no navegador

---

## Fase 1 — Estrutura HTML

* Criar a estrutura principal do layout
* Implementar a seção de estatísticas
* Adicionar os três cards (Total, Ativas, Completas)

---

## Fase 2 — Estilização Base

* Aplicar reset global
* Configurar variáveis de cores
* Estilizar body e container

---

## Fase 3 — Cards de Estatísticas

* Estilizar os cards
* Aplicar efeito visual (glassmorphism)
* Implementar animações de hover

---

## Fase 4 — Área de Adição de Tarefas

* Estilizar campo de input
* Estilizar botão de adicionar
* Ajustar layout com flexbox

---

## Fase 5 — Filtros

* Estilizar botões de filtro
* Implementar estado ativo
* Adicionar efeitos de interação (hover)

---

## Fase 6 — Lista de Tarefas (UI)

* Criar estrutura visual das tarefas
* Estilizar tarefas ativas
* Estilizar tarefas concluídas

---

## Fase 7 — Estado da Aplicação

* Criar estrutura de dados para armazenar tarefas
* Definir estado inicial da aplicação

---

## Fase 8 — Adicionar Tarefas

* Implementar função de criação de tarefas
* Validar entrada do usuário
* Limpar campo após inserção

---

## Fase 9 — Renderização

* Implementar renderização dinâmica das tarefas
* Atualizar o DOM corretamente

---

## Fase 10 — Concluir Tarefas

* Implementar marcação de tarefa como concluída
* Atualizar estado e interface

---

## Fase 11 — Remover Tarefas

* Implementar remoção de tarefas
* Atualizar lista após remoção

---

## Fase 12 — Filtros Funcionais

* Implementar lógica de filtragem
* Integrar filtros com renderização

---

## Fase 13 — Contadores

* Calcular total de tarefas
* Calcular tarefas ativas
* Calcular tarefas concluídas
* Atualizar valores na interface

---

## Fase 14 — Eventos de Interação

* Implementar clique no botão de adicionar
* Implementar envio com tecla Enter

---

## Fase 15 — Responsividade

* Ajustar layout para telas menores
* Melhorar usabilidade em dispositivos móveis

---

## Fase 16 — Refinamento Final

* Melhorar interface visual
* Ajustar animações
* Revisar experiência do usuário
